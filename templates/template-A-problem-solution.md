# Template A — Hook → Problem → Solution → Twist → CTA

General-purpose teaching/value content. Use for **evergreen explainers** (not tied to a
single day's news).

## Constraints
- ≤ 60s → ~130–150 words. First 3 seconds = the hook.
- Twist = the non-obvious insight (the original value). Cite sources if factual claims.
- End with a CTA. Append `#Shorts`.

`{ script_body, caption, hashtags[] }`.
